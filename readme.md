# Videoplus

> Videoplus es una aplicación de gestión de vídeos en la que podrás ver y compartir videos a través de la web.

## Instalación

Instala las dependencias de la aplicación

```bash
npm install
```

Iniciar servidor de pruebas

```bash
npm start
```
